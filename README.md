## 🧭 What Is ANOMS-Hub?

ANOMS-Hub is the **meta-repository** for the entire Anom Originals (AO) ecosystem. It houses documentation, governance policies, architectural decisions, brand guidelines, contributor resources, and the project roadmap — everything that ties the AO universe together.

This repo doesn't contain application code. Instead, it serves as the **north star** that connects and coordinates all other ANOMS repositories.
